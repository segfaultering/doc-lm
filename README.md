This project aims to create a chatbot-esque agent that has the simple functionality of taking in files as input and then answering questions that the user has about those files. The scope of the project is limited intentionally as this is more of an incremental project where I will be making things more and more elaborate and complex as time goes on, while the initial versions of this project will definitely remain simple.

## Technologies:
- SQLite3: For the local database. 
- ChromaDB: For the vector database.
- Groq: For the LLM. 
- Gemini: For the embeddings.   
- Gradio: For the UI.
- Langchain: For the model orchestration.

Things may (will) change as the project evolves.

Other dev dependencies include:
- Pytest: For the testing framework.
- Ruff: For the formatting and linting.
- Mypy: For the static code analysis.

The README.md will continue to get updated as things change and move around.
